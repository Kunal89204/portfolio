# Overview

Admatrica is a marketing platform built for a European client, focused on showcasing AI-driven marketing solutions and providing an engaging experience for potential customers. The project required a modern, high-converting website that effectively communicated the client's services while maintaining strong performance and responsiveness across devices.

# My Role

Frontend Developer

I was responsible for developing the user-facing experience, implementing responsive layouts, integrating third-party services, and ensuring a smooth and engaging user journey.

# Key Features

- Modern marketing-focused landing pages
- Fully responsive design for desktop and mobile devices
- AI-powered voice chatbot integration using ElevenLabs
- Automated conversation delivery via email
- Fast page loading and optimized user experience
- SEO-friendly frontend architecture

# Tech Stack

- Next.js
- React
- Tailwind CSS
- ElevenLabs
- Resend

# Challenges

## AI Chatbot Integration

One of the key requirements was integrating an AI-powered voice assistant using ElevenLabs. The chatbot needed to provide an interactive experience while maintaining seamless communication with potential customers.

## Conversation Tracking

To ensure no lead was missed, every chatbot interaction was captured and automatically forwarded to the client via email using Resend. This enabled the client to review conversations and follow up with potential customers efficiently.

## Performance & Responsiveness

As a marketing website, fast loading times and responsive layouts were critical. Special attention was given to optimizing the frontend experience across various screen sizes and devices.

# Results

- Successfully delivered a production-ready marketing platform
- Implemented AI voice interactions to improve visitor engagement
- Automated lead communication through email notifications
- Created a responsive and modern user experience
- Built a scalable frontend foundation using Next.js

# What I Learned

This project strengthened my experience in building marketing-focused web applications, integrating AI-powered services, and creating automated workflows that connect user interactions directly with business processes.
